## Getting Started

You can setup this project locally

### Prerequisites

* You will need to have Docker Desktop installed and running on your computer


### Installation

1. Make sure you have docker up and running. 
2. Inside this repository root run
   ```sh
   docker-compose up -d
   ```

### Teardown

   ```sh
   docker-compose down
   ```
